Site Plan Document

Site Name

Samuel Casellas

This name was selected to represent the main subject featured in the software portfolio.

Optional domain availability: samuelcasellas.me

Site Purpose

This site will be a showcasing for the following on the subject:

  • Background information regarding schooling, profession, field of study, etc.
  • Executive summary
  • Significant projects completed
  • Point of contact to reach out to the author
  • Facilitating online purchases of our products

Scenarios

  1. What sort of experience does this person have?
  2. Has this person built a significant application?
  3. Does this candidate fit the job description?

Color Schema

  • Primary Color: #4A90E2 (Blue) - Used for headings, buttons, and accents
  • Secondary Color: #2ECC71 (Green) - Used for highlights, icons, and secondary buttons
  • Background Color: #F5F5F5 (Light Gray) - Used for the main background
  • Text Color: #333333 (Dark Gray) - Used for body text

Typography

  • Headings: Roboto Slab - A clean, modern serif font for titles and headings
  • Body Text: Open Sans - A highly legible sans-serif font for main content
  • Accents: Montserrat - A bold sans-serif font for buttons and call-to-action elements
